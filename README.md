# CPI PREDICTION USING-MACROECONOMIC INDICATORS AND MACHINE LEARNING MODEL
This repository hosts the data, scripts, and documentation for my Final Year Project. The project focuses on synthesizing macroeconomic determinants with machine learning architectures to forecast the Consumer Price Index (CPI) in Malaysia.

Dataset & Variables
The study utilizes monthly time-series data spanning from January 2013 to January 2026. The following macroeconomic indicators are used as features to predict CPI:

- PPI (Producer Price Index): Captures cost-push inflation from the production side.

- OPR (Overnight Policy Rate): Reflects the impact of Bank Negara Malaysia’s monetary policy.

- M3 Money Supply: Measures broad liquidity within the economy.

- Unemployment Rate: Provides insight into the Phillips Curve relationship and labor market pressure.

- Exchange Rate: Accounts for the impact of currency fluctuations on imported goods and services.
